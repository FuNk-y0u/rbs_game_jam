The RBS Game Jam is an internal game development competition organized by the Robotics and Programming Club of Rato Bangala School.
This two-week event provides participants with an opportunity to showcase their skills and creativity, either individually or as part of a team.
Contestants should build a game related to the theme provided, and closely integrate it to their games. Games will be judged from different perspectives: gameplay, artstyle, creativity, theme.

Rules

1. Can use any engine or game possible
2. Should be able to run on windows
3. Should not have malicious code in your game
4. Should relate to the theme provided
5. Should include your group members' names in the game as text.
6. A team should have only 4 members max.
7. You can join in as individual also!
