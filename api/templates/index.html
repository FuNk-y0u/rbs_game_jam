About Game Jam

The RBS Game Jam is an internal game development competition organized by the Robotics and Programming Club of Rato Bangala School. This two-week event provides participants with an opportunity to showcase their skills and creativity, either individually or as part of a team. Contestants should build a game related to the theme provided, and closely integrate it to their games.

Rules

1. You may use any game engine or platform of your choice.
2. The game must be compatible with Windows operating systems.
3. The game should be free of any malicious code or harmful software.
4. The game must adhere to the provided theme.
5. The names of all group members must be included within the game as text.
6. Each team may consist of a maximum of four members.
7. Participation as an individual is also permitted.
